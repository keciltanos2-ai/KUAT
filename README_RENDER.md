Deployment & ENV instructions
-----------------------------
1. Set environment variables (Render / hosting):
   GROQ_API_KEY=your_groq_api_key
   MODEL=llama3-70b-8192
   YOUTUBE_VIDEO_ID=YouTubeVideoID
   SOCIAL_INSTAGRAM=https://www.instagram.com/p/POST_PERMALINK
   SOCIAL_TWITTER=https://twitter.com/user/status/ID
   SOCIAL_FACEBOOK=https://www.facebook.com/page/posts/ID
   OPENWEATHER_API_KEY=your_openweather_key
   PORT=10000

2. npm install
3. npm start
